---
title: "Get Zoom OAuth Connection"
api: "GET /api/v1/zoom_oauth_connections/{object_id}"
description: "Gets a Zoom OAuth connection."
---

## Authentication

Requires API key authentication via the `Authorization` header.

## Path Parameters

<ParamField path="object_id" type="string" required>
  Zoom OAuth Connection ID (e.g., `zoc_abcdef1234567890`)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the Zoom OAuth connection
</ResponseField>

<ResponseField name="zoom_oauth_app_id" type="string">
  ID of the Zoom OAuth app this connection belongs to
</ResponseField>

<ResponseField name="user_id" type="string">
  Zoom user ID
</ResponseField>

<ResponseField name="account_id" type="string">
  Zoom account ID
</ResponseField>

<ResponseField name="state" type="string">
  Connection state (`connected` or `disconnected`)
</ResponseField>

<ResponseField name="metadata" type="object">
  Metadata associated with the Zoom OAuth connection
</ResponseField>

<ResponseField name="is_local_recording_token_supported" type="boolean">
  Whether the connection supports generating local recording tokens
</ResponseField>

<ResponseField name="is_onbehalf_token_supported" type="boolean">
  Whether the connection supports generating onbehalf tokens
</ResponseField>

<ResponseField name="connection_failure_data" type="object">
  Information about connection failures, if any
</ResponseField>

<ResponseField name="created_at" type="string">
  Timestamp when the connection was created (ISO 8601 format)
</ResponseField>

<ResponseField name="updated_at" type="string">
  Timestamp when the connection was last updated (ISO 8601 format)
</ResponseField>

<ResponseField name="last_successful_sync_at" type="string">
  Timestamp of the last successful sync (ISO 8601 format)
</ResponseField>

<ResponseField name="last_attempted_sync_at" type="string">
  Timestamp of the last attempted sync (ISO 8601 format)
</ResponseField>

<RequestExample>

```bash cURL
curl --request GET \

  --header 'Authorization: Token YOUR_API_KEY_HERE' \

```
