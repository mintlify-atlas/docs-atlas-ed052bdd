---
title: "Delete Calendar"
api: "DELETE /api/v1/calendars/{object_id}"
description: "Permanently deletes a calendar and all associated data."
---

## Authentication

Requires API key authentication via the `Authorization` header.

## Path Parameters

<ParamField path="object_id" type="string" required>
  Calendar ID (e.g., `cal_abcdef1234567890`)
</ParamField>

## Response

Returns a `200` status code on success with no response body.

<RequestExample>

```bash cURL
curl --request DELETE \
  --url https://app.attendee.dev/api/v1/calendars/cal_abcdef1234567890 \
  --header 'Authorization: Token YOUR_API_KEY_HERE' \
  --header 'Content-Type: application/json'
```

</RequestExample>

<ResponseExample>

```json Response
{}
```

</ResponseExample>
